## Summary

### Candidate effect: zero (Δ(C-T) = 0.00 avg)

The key question is whether v0.12.2's mainline concern pass changes benchmark scores relative to the same baseline under the same prompt conditions. **It does not.** Arm C (candidate + terse) and Arm B (baseline + terse) score identically at 42.50 avg.

Per-model Δ(C-T):
- Opus: 0 — unaffected
- Sonnet: −1 — within noise; attributable to slightly less specific FM1 in Arm C
- GPT-5.4: 0 — unaffected; Step 5c.3 compaction did not impair analysis
- Codex: +1 — slight benefit; shorter spec marginally easier to trace under terse constraint

**Verdict: No regression. The candidate is safe to merge.**

### Terse effect: moderate (Δ(B-A) = −1.00 avg)

The terse overlay itself has a meaningful negative effect on GPT-family models:
- GPT-5.4: −2 (Failure Modes and Loop Start depth reduced)
- Codex: −2 (Sequence and Loop Start compressed)
- Opus: 0 (unaffected — already structured)
- Sonnet: 0 (unaffected — natural response style matches terse constraint)

This is a known pattern: the terse overlay trades analytical depth for conciseness in GPT-family models. It is not a concern for production use since the overlay is a benchmark instrument, not a runtime constraint.

## Cross-Arm Convergence

Findings that appear in all 3 arms (12/12 runs) are structural characteristics of the v0.12.x architecture, not artifacts of the candidate spec or terse constraint:

1. **Intent Router misclassification** — primary loop-start failure mode in all 12 runs
2. **Step 5c render pipeline complexity** — flagged as top failure mode in all 12 runs; unchanged by spec compaction
3. **MFA Clarity ceiling at 7/10** — hard invariant physically separated from routing entry; no urgency label; no START HERE anchor
4. **Continuation path absorption** — secondary loop-start cause in 9/12 runs

New finding from Arm C (not seen in Arms A or B):
- **Step 3b optional-vs-gated contradiction** — 2/4 Arm C models independently flagged the tension between Step 3b's optional plan file and Step 3c's hard gate on its absence. This exists in the baseline; its appearance in Arm C may reflect increased attention caused by the shorter spec making the 3b/3c interaction more visible.
